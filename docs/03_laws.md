# 三大核心定律

## 1. 稳定律

系统的稳定性 \(\propto\) 结构复杂度 \( \times \) 环境压力的匹配度。

*注：过高或过低的复杂度相对于环境，均会导致失稳。*

## 2. 清除陷阱（理性短视律）

为提升短期效率而清除系统内的“当前冗余”（如备用资源、多样化尝试），虽符合短期最优逻辑，但会牺牲系统的长期韧性与适应力。

*这是系统常犯的“理性但致命”的策略错误。*

## 3. 相变点：同频状态

当系统内部涌现出足以覆盖（或劫持）个体欲望-恐惧的 **“元欲望”**（如民族存亡、信仰狂热、共同使命）时，系统将进入短暂而强大的 **“同频”** 状态。

此状态下，个体局部理性让位于系统整体目标，实现超越常规逻辑的协作与能量释放。
